## Overview

**ITVvf** is a minimal **Secure WebSocket (wss://) client implemented in C++**.  
It demonstrates how to establish a TLS connection, perform a WebSocket handshake, and exchange WebSocket text frames without relying on high-level WebSocket libraries.

---

## Features

- Secure WebSocket (`wss://`) client
- TLS handled using OpenSSL
- Manual WebSocket handshake
- Manual WebSocket frame encoding and decoding
- Client-side masking
- Interactive command-line interface
- Concurrent receive loop using `std::thread`
